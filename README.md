# Marketing Funnel & Conversion Performance Analysis (Power BI)

## 📌 Overview
This project analyzes marketing campaign data to understand how customers move through the funnel:
**Contacted → Interested → Converted**.

The goal is to identify drop-off points, evaluate conversion performance, and provide actionable business insights.

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- CSV Dataset (Bank Marketing Dataset)

---

## 📊 Key Metrics
- Total Customers Contacted: 45,000+
- Converted Customers: ~5,000
- Conversion Rate: ~11.7%
- Average Campaign Calls: 2.76

---

## 📈 Dashboard Features
- Funnel Visualization (Customer Journey)
- KPI Cards (Key Metrics)
- Conversion Distribution (Donut Chart)
- Conversion by Age Group
- Conversion by Month
- Conversion by Job Type

---

## 🔍 Key Insights
- Significant drop-off occurs at the final conversion stage
- Only ~11.7% of contacted customers convert
- Age group 25–34 has the highest conversion rate
- Management and technician roles perform best
- Campaigns in May show peak performance

---

## 💡 Recommendations
- Focus on high-converting age groups (25–44)
- Improve follow-up strategies to increase final conversion
- Avoid excessive repeated calls
- Target high-performing months for campaigns

---

## 🚀 Outcome
Built a professional marketing analytics dashboard to support data-driven decision-making and improve conversion strategies.
